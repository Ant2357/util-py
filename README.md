# util-py
日常生活で使う小さな Python コードの置き場

## Usage
※ 事前に ffmpeg を導入してください。

```sh
git clone https://github.com/Ant2357/util-py.git
cd util-py

# 説明簡略化の為に書いてないけど、基本は venv 等を使ってください
pip install -r requirements.txt
```
